Games

Here you can find mostly simple games that I've created during either game jams or my process of learning game development and smaller projects I've completed when I needed a break from my bigger projects.

The games are in a more or less polished state and all of them still have a lot of room to improve. So if you like any game in particular let me know. If there will be enough interest for a certain game I'm more likely to put more time into it and make it better. Each game has a section where I detail what the future holds for them.
