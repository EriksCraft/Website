What is my main project?

It's a story driven 2D pixel art RPG game. For now it's called project BodyFrame, but we'll see about that, we can say it's a code name. Yeah, that sounds cool.

Fun fact: my "artistic name" BuckySide came from the first village in project BodyFrame, I don't remember how I came up with it, but I liked it ever since.

I can't reveal much about the story, as it's the focus of the game, but I'm happy to share the mechanics that will be involved in the game, all of the graphics and of course the features.
